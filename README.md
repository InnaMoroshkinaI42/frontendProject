# GOOD4ME — Landing Page

Лендинг для бренда витаминов и пищевых добавок GOOD4ME. Проект собран на **Vite** с использованием чистого HTML/CSS и JavaScript-модулей.

## Старт
### Требования
- Node.js (версия 18 или выше)
- npm / yarn / pnpm

### Установка и запуск
    ```bash
    git clone <ссылка-на-репозиторий>
    cd good4me-landing
    npm install
    npm run dev

### Структура
src/index.html — разметка страницы
src/styles/ — стили (темы, токены)
public/ — статические файлы (картинки, иконки)
src/index.js — JS‑логика (мобильное меню и т. д.)
